 Personalized Walking Time Recommendation System

An AI-powered health recommendation web application that predicts stress levels and recommends personalized walking durations based on user health parameters.

Features

- Stress Level Prediction
- Walking Time Recommendation
- Heart Rate Detection
- Weather Integration
- Step Tracker
- Health Chatbot
- User Authentication
- MySQL Database

 Technologies used

- Python
- Streamlit
- Random Forest
- HistGradientBoosting
- HTML
- CSS
- 
 Project Highlights:

🔹 Predicts user stress levels from health parameters such as sleep duration and heart rate.
🔹 Recommends optimal walking duration to improve physical and mental well-being.
🔹 Provides real-time health insights, calorie estimation, and fitness tracking.
🔹 Integrates weather conditions to ensure safe and comfortable walking plans.
🔹 Includes an AI-powered chatbot for personalized health guidance
- MySQL
- OpenCV
- Pandas
- NumPy
